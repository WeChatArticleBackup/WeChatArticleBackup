<h1 align="center">WeChat Article Backup</h1>
<p align="center"> 一些微信公众号文章的备份 </p>

## 🎈 Usage <a name="usage"></a>

* ```*.webarchive``` : 此文件可以在 iOS 、macOS等设备上打开，保留了网页所有信息，还原最原始的阅读体验。

* ```*.mhtml``` : 此文件可以在 windows 上用 Edge 、Chrome 等浏览器打开，保留了网页所有信息，还原最原始的阅读体验。

* ```*.jpg``` : 部分文章带有评论。
